# graph-visualizer-
Graph Visualizer (Python GUI Application)

A step-by-step, multi-window Graph Visualizer built using Python, Tkinter, Pandas, and Matplotlib.
This application allows users to upload CSV files, select graph types, choose columns, visualize data, and save graphs as images, all through an attractive full-screen GUI.

1. Features
Upload CSV file using file dialog
Multi-window guided interface (step-by-step)
Supports Line, Bar, and Scatter plots
 Dynamic column selection (X-axis & Y-axis)
Save generated graphs as PNG images
Full-screen responsive GUI
Converted into standalone Windows EXE
 No command-line interaction required for end users

2.Technologies Used
Python 
Tkinter – GUI development
Pandas – Data handling
Matplotlib – Data visualization
PyInstaller – EXE generation
